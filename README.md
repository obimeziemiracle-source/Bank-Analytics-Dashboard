# Bank-Analytics-Dashboard
This project is a Bank Anaytics Dashboard built in Power BI which is designed to help monitor customer accounts, loan performance, account balances, and customer behaviour across different cities.

## Business Problem
Banks need a simple way to track customer activities and identify trends that support better decision-making.

## Objectives
To develop an interactive dashboard that provides insights into customer demographics, account types, loan status, account balances, and regional customer behaviour.

## Tools Used
- Microsoft Excel
- SQL
- Power BI

##  Dashboard Features
- KPI Cards
- Customer Demographics
- Account Balance Analysis
- Loan Analysis
- Interactive Charts
- Slicers and Filters

## Key Insights
- Monitor account type distribution
- Compare customer behaviour across different cities.
- Track loan status across customers.
- Analyze customer balances.
